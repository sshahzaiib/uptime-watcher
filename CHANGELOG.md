# Changelog

## [1.0.0] - 2026-02-02

### Added
- **Dynamic Service Management**: Add, edit, and remove unlimited TCP/HTTP services.
- **Accessory Mode**: App runs stealthily in the background (hidden from Dock and Cmd+Tab).
- **Adaptive Icons**: Alternate "Check/Cross" icons that adapt to macOS Light/Dark mode.
- **Liquid Glass UI**: Modern, frosted-glass interface.
- **Persistence**: All settings and services are saved automatically.
- **Configurable Intervals**: Set check frequency from 10s to 1hr.
- **Minimize to Tray**: Closing the window hides it to the tray instead of quitting.

### Fixed
- Fixed immediate tray icon updates when changing settings.
- Fixed header text clipping in the UI.
- Corrected application architecture (removed unused lib.rs template code).
